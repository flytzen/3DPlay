Samle STL manipulator

This sample loads a sample STL file and allows you to move it etc.. It currently uses a Perspective camera, but an Ortographic camera would be better for this purpose. Etc, etc...

  • Use arrow keys to move the object (moving on the z-axis not implemented yet)
  • Drag the mouse to rotate
  • Use two finger swipes to change the size of the object (mouse wheel, middle button etc not implemented)
